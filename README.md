## 🎮 A Cybersecurity Themed Multiplayer Puzzle Game

Hack other players, solve encryption puzzles, upgrade your vault, and become the ultimate hacker!

![Game Screenshot](https://via.placeholder.com/800x400?text=Hacking+Puzzle+Game+Screenshot)

## ✨ Features

- 🔐 **User Authentication** - Register/Login with JWT tokens + Supabase Auth
- 💰 **Vault System** - Upgradeable security levels with coins
- 🧩 **Multiple Puzzle Types**:
  - Caesar Cipher
  - Binary Code Breaking
  - Pattern Recognition
  - Logic Riddles
  - Vigenère Cipher
- ⚔️ **Attack System** - Hack other players by solving puzzles
- 🏆 **Leaderboard** - Compete with global players
- ⭐ **XP & Leveling** - Earn XP, level up, unlock features
- 🎨 **Hacker Theme** - Dark terminal-style UI with glitch effects
- ☁️ **Supabase Integration** - Cloud database with real-time updates
